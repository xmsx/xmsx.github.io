Python简介

大神（吉多·范罗苏姆）闲着没事在圣诞节（1989）瞎编的的一门语言，结果火了，就这样。

特点

  • 语法简单
  • 通用性强
  • 丰富的库
  • 格式规范（适合有强迫症的）

环境搭建

Windows下在Python官网下载你要学习的版本，直接默认安装在电脑上。在windows下运行Python要配置环境变量，win10要在系统环境变量path里添加 C:\Python27 和 C:\Python27\Scripts 两个路径，千万别忘了分号。然后在命令行输入python即可看到python的版本，如图：

同时显示 >>> 表示进入了python环境，输入exit() 可退出环境。
